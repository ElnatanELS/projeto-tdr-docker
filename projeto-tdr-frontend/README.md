# [Projeto Tempo de Reação]

> Projeto de pesquisa: Tempo de Reação do lado ipisilateral a lesão em indivíduos hemiparéticos pós acidente vascular encefálico



## Instalação
```sh
npm install
```


## Execução

*1 rodar o banco
```sh
npm run mongo
```

*2 roda o NodeJS

```sh
npm run nodemon
```

*2 roda o NodeJS

```sh
ng serve
```

Obs.: rodar em terminais separados
